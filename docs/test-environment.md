# Test Environment

## Browsers

Testing will be performed on the following browsers via [BrowserStack](https://www.browserstack.com):

- **Chrome**
- **Firefox**
- **Safari**
- **Edge**

## Data

Mock data is provided via the following [JSONPlaceholder](https://jsonplaceholder.typicode.com/) endpoints:

- `/users`
- `/posts`
- `/comments`
